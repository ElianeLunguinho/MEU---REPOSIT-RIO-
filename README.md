## 🚀 Como o Projeto Foi Construído

Este portfólio foi desenvolvido com foco em **presentação profissional**, **responsividade** e **experiência do usuário**. Abaixo está a descrição de como cada componente foi implementado:

### 1. **Estrutura HTML**
- **index.html**: Estrutura principal do site com seções semânticas (header, nav, sections, footer)
- Utilização de tags semânticas para SEO e acessibilidade
- Integração com Font Awesome para ícones
- Google Fonts para tipografia personalizada

### 2. **Estilização CSS**
- **style.css**: Estilos globais e específicos para cada seção
- Design responsivo com Media Queries
- Animações e transições suaves
- Cores e tipografia profissional
- Layout flexbox e grid

### 3. **Interatividade JavaScript**
- **script.js**: Funcionalidades dinâmicas do site
- Menu mobile toggle
- Animações de scroll
- Chatbot interativo (LunguinhoBot)
- Validações de formulários
