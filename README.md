# Portfolio - Eliane Lunguinho

Este é o portfólio profissional de Eliane Lunguinho, desenvolvedora em formação especializada em Front-End e Cloud AWS.

## Estrutura do Projeto

```
portfolio/
├── index.html          # Estrutura principal do site
├── style.css           # Estilos CSS
├── script.js           # Funcionalidades JavaScript
├── images/             # Imagens do portfólio
├── projetos/           # Arquivos dos projetos
└── README.md           # Este arquivo
```

## Seções do Portfólio

1. **Home** - Pitch profissional e chamada para ação
2. **Minha Jornada** - Storytelling da transição de carreira
3. **Sobre** - História profissional
4. **Habilidades** - Skills técnicas e soft skills
5. **Projetos** - 4 projetos principais
6. **Experiência & Formação** - Credenciais profissionais
7. **Contato** - Links para redes sociais e email
8. **LunguinhoBot** - Chatbot interativo

## Como Usar

1. Clone este repositório
2. Adicione suas imagens na pasta `images/`
3. Atualize os links dos projetos nas seções correspondentes
4. Personalize o conteúdo com suas informações
5. Faça o deploy usando GitHub Pages, Netlify ou Vercel

## Tecnologias Usadas

- HTML5
- CSS3
- JavaScript
- Font Awesome (ícones)
- Google Fonts

## Deploy

### GitHub Pages
1. Vá para Settings > Pages
2. Selecione a branch main
3. Salve as configurações

### Netlify
1. Arraste a pasta do projeto para o Netlify
2. O site será publicado automaticamente

### Vercel
1. Instale o CLI do Vercel
2. Execute `vercel deploy`
